# Star-Patterns-2

This is a java program to output star patterns when input the number of lines of the pattern

Pattern of this program:            
                                      
                            * 
                            * * 
                            * * * 
                            * * * * 
                            * * * * * 
                            * * * * 
                            * * * 
                            * * 
                            * 
                                      
                                      
This is the output of five lines

You can enter number of lines as you wish
